Getting Started

New to TeamBeat? Here are some quick tips to get you started:

  • Create a Board Series: Start by creating a board series to organize related boards
  • Add Team Members: Invite your team members with appropriate roles (admin, facilitator, or member)
  • Create Boards: Add boards for different sessions like retrospectives, planning, or brainstorming
  • Real-time Collaboration: All changes are synced instantly across all connected users
  • Voting System: Use the built-in voting to prioritize ideas and make decisions

Common Issues

Can't see real-time updates?

Make sure your browser supports WebSockets and that you're not behind a restrictive firewall. Try refreshing the page if updates aren't appearing.

Votes not counting?

Check that you haven't exceeded your vote allocation for the board. Board facilitators can adjust vote allocations in the board settings.

Can't access a board?

Ensure you've been added to the board series with appropriate permissions. Contact your team's board administrator if you need access.

Cards not saving?

Verify your internet connection is stable. Cards are saved automatically, but connection issues can prevent saving. Try refreshing and re-entering your content.

Features Overview

Board Types

  • Retrospective Boards: Perfect for team retrospectives with customizable columns
  • Brainstorming Boards: Open collaboration with voting to identify top ideas
  • Planning Boards: Organize and prioritize tasks or user stories

User Roles

  • Admin: Full control over series, boards, and member management
  • Facilitator: Can create/edit boards and manage sessions
  • Member: Can participate in boards they have access to

Privacy Features

  • Blame-Free Mode: Hide card authors to encourage open feedback
  • Private Series: Control who can access your team's boards
  • Session Control: Facilitators can control when voting opens/closes

Browser Requirements

TeamBeat works best with modern browsers that support:

  • Server-Side Event streaming for real-time updates
  • Local Storage for session management
  • Modern JavaScript (ES2020+)

Recommended browsers:

  • Chrome 90+
  • Firefox 88+
  • Safari 14+
  • Edge 90+

Account Management

Changing Your Password

  1. Go to your Profile page
  2. Click "Change Password"
  3. Enter your current password and new password
  4. Click "Update Password"

Updating Profile Information

You can update your display name and other profile information from the Profile page. Note that your email address is used for login and notifications.

Troubleshooting

Performance Issues

If TeamBeat is running slowly:

  • Close other browser tabs to free up memory
  • Disable browser extensions that might interfere
  • Clear your browser cache and cookies
  • Try using an incognito/private browsing window

Connection Problems

If you're having trouble connecting:

  • Check your internet connection
  • Disable VPN if you're using one
  • Try a different network (mobile hotspot)
  • Contact your IT department about firewall restrictions
